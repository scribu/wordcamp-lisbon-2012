## Setup

Have node.js and npm installed.

```bash
# Install CoffeeScript (and cake)
npm install -g coffee-script

# Install dependencies
cd slides
npm install

## Usage

```bash
# Build HTML page
cake build

# View slides
firefox slides/index.html
```
